## Introduction to Xandeum Validator
Xandeum Foundation DevNet is based on the Agave codebase v2.2.0.

## Installation and Configuration Steps
To install and configure a Xandeum validator, complete the following steps in order:
1. Old Server housekeeping
2. Access your server
3. Ports Setup
4. Setup Your Disks
5. Validator Installation
6. Setup System Service
7. Setup LogRotate
8. Starting your validator
9. Monitoring your validator
10. Onboarding
11. Validator Commands
12. Learn Tmux
13. Troubleshooting

## Updating a Running Validator
To update a running validator, follow the instructions for **Validator UPDATE version**.
