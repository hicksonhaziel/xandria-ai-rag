## Technical Details
The Xandeum network uses a Merkel tree to store data, which allows for efficient and secure data storage. The network also includes an optional encryption layer to protect data from unauthorized access. The PNodes store data in chunks, called pages, which are redundantly stored across the network to ensure data availability.

## Merkel Proofs
A Merkel proof is a cryptographic proof that allows users to verify the integrity of data stored on the network. It works by creating a hash tree, where each node in the tree represents a hash of the data. This allows users to verify that the data has not been tampered with or altered.
